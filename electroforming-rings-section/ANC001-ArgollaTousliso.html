¿Que es el electroformado?

El electroformado es un proceso donde a una pieza de resina retocada se le aplica una pintura conductora para introducirse en un baño de electrolisis donde se adhiere un laminado de plata ley 0.999. Ya una vez terminado, se pule, se le da brillo y se laquea para obtrener como resultado una pieza voluminosa y un poco mas ligera a comparacion de un proceso de vaciado.
